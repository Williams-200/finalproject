Lesson 1

What is a Function?

A function is a named block of code that only runs when you call it. Think of it like a recipe — you write the recipe once, and you can cook the same dish anytime by just saying its name!

📱

Like a phone shortcut

You press one button and it dials a long number. The shortcut hides the complexity!

🍕

Like a food recipe

Write the recipe once, follow it any time you want pizza. No need to reinvent it!

🎮

Like a game ability

Press a button → character jumps. The jump code runs every time without rewriting it.

In Python, a function helps you:

  • 🚫 Avoid repeating the same code over and over
  • 📦 Organise code into neat, named blocks
  • 🔄 Reuse code anywhere in your program
  • 🐛 Fix bugs in one place instead of many
💡

Python has many built-in functions you already know — like print(), len(), and range(). Now you'll learn to make your own!

Lesson 2

Defining & Calling a Function

To define (create) a function in Python, use the def keyword. To call it (run it), just write its name followed by parentheses ().

def greet () :
← def keyword + function name
print ( "Hello, World!" )
← indented code block (the recipe)
greet ()
← calling the function runs it!
🐍 Define and Call a Function
# Step 1: DEFINE the function
def greet():
    print("Hello, Python student!")
    print("Welcome to functions!")

# Step 2: CALL the function (as many times as you want!)
greet()   # Hello, Python student!
greet()   # Hello, Python student!
greet()   # Hello, Python student!

📏 Naming Rules (same as variables): start with a letter or underscore, use only letters/numbers/underscores, no spaces, case-sensitive. Use snake_case for Python functions!

Lesson 3

Arguments & Parameters

You can send information into a function using arguments. Inside the function definition, the placeholder names are called parameters.

Function Definition
def greet(name):
Parameter 📥

The placeholder variable — it receives the value

→
Function Call
greet("Alice")
Argument 📤

The actual value you pass in

🐍 One Argument
def greet(name):
    print("Hello,", name)

greet("Alice")   # Hello, Alice
greet("Bob")     # Hello, Bob
greet("Maria")   # Hello, Maria
🐍 Multiple Arguments
def describe_student(name, grade, subject):
    print(name, "is in grade", grade)
    print("Favourite subject:", subject)

describe_student("Alice", 8, "Math")
🐍 Default Parameter Values
# Give a parameter a DEFAULT value
def greet(name = "Student"):
    print("Hello,", name)

greet("Alice")  # Hello, Alice
greet()         # Hello, Student  ← uses default

Lesson 4

Return Values

Functions can send data back to the caller using return. Think of it like ordering at a restaurant — you give the order (arguments) and get a meal back (return value)!

👤

Caller

result = add(3, 4)
passes 3 and 4 →
← gets back 7
⚙️

Function

return 3 + 4
🐍 Basic return
def add(x, y):
    return x + y

result = add(3, 4)
print(result)    # 7

# Use the result directly
print(add(10, 20))  # 30
🐍 Practical Example — Temperature Converter
def celsius_to_fahrenheit(celsius):
    return (celsius * 9 / 5) + 32

print(celsius_to_fahrenheit(0))    # 32.0
print(celsius_to_fahrenheit(100))  # 212.0
print(celsius_to_fahrenheit(37))   # 98.6

Lesson 5

Lambda Functions

A lambda function is a tiny, anonymous (unnamed) function written in a single line. Perfect for quick, one-off calculations!

Syntax: lambda parameters : expression

Normal Function
def double(x):
    return x * 2

print(double(5))  # 10
⚡
Lambda Function
double = lambda x: x * 2

print(double(5))  # 10
🐍 Lambda with Multiple Arguments
# Add two numbers
add = lambda x, y: x + y
print(add(3, 4))         # 7

# Calculate area of rectangle
area = lambda w, h: w * h
print(area(5, 3))        # 15

# Lambda inside a function (multiplier maker)
def make_multiplier(n):
    return lambda x: x * n

triple = make_multiplier(3)
print(triple(7))           # 21

Lesson 6

Variable Scope

Scope means where a variable is visible and accessible. Variables created in different places have different scopes — they don't always see each other!

🌍 Global Scope
x = "global"
🏠 Local Scope (inside function)
x = "local"
This x is different from the global x!
🐍 Local Scope — variable only lives inside the function
def my_func():
    secret = "I'm local!"   # local variable
    print(secret)            # works here ✅

my_func()
print(secret)   # ❌ ERROR! 'secret' doesn't exist outside
🐍 Global Scope — accessible everywhere
greeting = "Hello!"   # global variable

def say_hi():
    print(greeting)   # ✅ can read global variable

say_hi()              # Hello!
print(greeting)     # Hello!
🐍 Using the global keyword to modify from inside
count = 0

def increment():
    global count      # declare we want the global one
    count += 1

increment()
increment()
print(count)   # 2
