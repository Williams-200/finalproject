Lesson 1

What are Conditionals?

Every day you make decisions based on conditions. "If it's raining, I'll take an umbrella." Python works the same way!

A conditional lets your program make decisions — it checks if something is True or False and then runs certain code.

🌧️

Real life: If it's raining → take an umbrella

🐍

Python: if raining == True:
→ run code

🎮

Real life: If your score > 100 → level up!

🐍

Python: if score > 100:
→ level_up()

💡

Python supports the standard conditions from maths: ==, !=, <, <=, >, >=. We'll learn all of them in the next lesson!

Lesson 3

The if Statement

An if statement checks a condition. If the condition is True, the indented code runs. If it's False, that code is skipped.

if condition :
→ code to run if True

⚠️ Indentation matters! Python uses spaces (not curly braces {}) to define which code belongs inside the if block. Always indent with 4 spaces or 1 tab.

🐍 Example — Is b greater than a?
a = 33
b = 200

if b > a:
    print("b is greater than a")  # This runs!
🐍 Multiple statements inside if
score = 95

if score >= 90:
    print("You passed!")
    print("Great job!")
    print("You got an A!")

Lesson 4

elif & else

Real programs need more than one path. elif (short for "else if") checks another condition if the first was False. else runs when none of the conditions were True.

if

First condition to check

if age < 13:
elif

Another condition (checked only if if was False)

elif age < 18:
else

Runs if none of the above were True

else:
🐍 Full if / elif / else Example
a = 33
b = 33

if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")   # This runs!
else:
    print("a is greater than b")
🐍 School Grade Example
score = 78

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")     # This runs!
else:
    print("Grade: F")

🐍 Logical Operators in Action
age = 15

# and — both must be true
if age > 12 and age < 18:
    print("You are a teenager!")   # Runs!

# or — at least one must be true
if age < 5 or age > 13:
    print("You are very young or a teen!")  # Runs!

# not — flips the result
if not age == 20:
    print("You are not 20!")   # Runs!

Lesson 6

Short-Hand if

Python lets you write a simple if statement in a single line. This is called short-hand or ternary syntax. It's great for quick checks!

Normal (multi-line)
if a > b:
    print("a wins")
⚡
Short-Hand (one line)
if a > b: print("a wins")

There's also a one-line if-else using the ternary operator — it returns a value based on a condition:

🐍 Ternary (One-Line if-else)
a = 10
b = 20

# Syntax: value_if_true if condition else value_if_false
winner = "a" if a > b else "b"
print(winner)   # b

# Practical example: pass or fail
score = 72
result = "Pass" if score >= 60 else "Fail"
print(result)   # Pass
🏆

Short-hand is handy for simple checks, but for complex logic always use the full multi-line format — it's much easier to read!
