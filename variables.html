Lesson 1

What is a Variable?

Think of a variable like a labeled box 📦. You write a name on the outside, and you put something inside it. Whenever you need what's inside, you just say the name of the box!

🔒
name
"Alice"
🐍 Python Example
name = "Alice"
age  = 13
height = 5.4
print(name)   # Alice
print(age)    # 13
💡

Key Fact: In Python you do not need to declare a variable before using it. A variable is created the moment you assign a value to it with the = sign.

🆕

Lesson 2

Creating Variables

Creating a variable is simple! You just type a name, then =, then the value you want to store.

variable_name = value
🐍 Python Example
# Storing a name (text)
student = "Maria"

# Storing a grade (number)
grade = 95

# Storing a GPA (decimal)
gpa = 3.75

# Variables can change!
grade = 98
print(grade)   # 98
🔄

Variables can change! You can reassign a variable at any time — just use = again with the new value. The old value is replaced.

Lesson 3

Data Types

Variables can store different kinds of data. The four most common types for beginners are:

💬

String str

Text surrounded by quotes

name = "Alice"
city = 'NYC'
🔢

Integer int

Whole numbers (no decimal)

age   = 13
score = 100
📐

Float float

Decimal numbers

gpa    = 3.9
height = 5.4
✅

Boolean bool

True or False only

is_student = True
has_pet   = False
🐍 Checking the Type with type()
x = 42
print(type(x))   # <class 'int'>

y = "hello"
print(type(y))   # <class 'str'>

z = 3.14
print(type(z))   # <class 'float'>

Lesson 4

Naming Rules

Python has specific rules for how you can name your variables. Break these rules and your program won't run!

✅
Start with a letter or underscore _ name = "ok"  |  _value = "ok"
❌
Cannot start with a number 1name = "error"
✅
Can contain letters, numbers, underscores student1 = "ok"  |  my_var = "ok"
❌
No spaces or special characters my-var = "error"  |  my var = "error"
⚠️
Case-Sensitive: uppercase ≠ lowercase Name, name, and NAME are three different variables!
❌
Cannot use Python keywords if = 5, print = "hi", for = 1

Lesson 5

Naming Styles

When a variable name has multiple words, Python programmers use one of these three styles. The most popular for Python is Snake Case!

🐪

camelCase

First word is lowercase. Each new word starts with a capital letter.

myVariableName
🦙

PascalCase

Every word starts with a capital letter.

MyVariableName
🐍

snake_case Recommended

Words are separated by underscores. All lowercase.

my_variable_name
🏆

The official Python style guide (PEP 8) recommends snake_case for variable names. Most Python programmers follow this!
