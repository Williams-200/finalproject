Lesson 1

What are Loops?

Imagine you had to print your name 100 times. Would you type it 100 times? Of course not! A loop lets Python repeat a block of code automatically — as many times as you want.

😩 Without a Loop
print("Hello")
print("Hello")
print("Hello")
print("Hello")
print("Hello")
# ... 95 more lines!
→
😎 With a Loop
for i in range(100):
    print("Hello")
# Done! Just 2 lines 🎉
📋

for loop

Repeats over a list, string, or number range — a fixed number of times.

⏳

while loop

Repeats as long as a condition is True — stops when it becomes False.

💡

Python has two types of loops: for and while. We'll master both!

📋

Lesson 2

The for Loop

A for loop iterates — it goes through each item in a sequence (a list, string, tuple, etc.) and runs the code block once for each item.

for item in sequence :
→ code to run for each item
🐍 Loop through a list
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry
🐍 Loop through a string
# Strings are sequences of characters!
for letter in "Python":
    print(letter)

# Output: P y t h o n

Lesson 3

The range() Function

range() generates a sequence of numbers. It's perfect when you want to repeat something a specific number of times or loop through numbers.

range(n)

Produces numbers from 0 up to (not including) n

range(5) → 0, 1, 2, 3, 4
range(start, stop)

Produces numbers from start up to (not including) stop

range(2, 6) → 2, 3, 4, 5
range(start, stop, step)

Same, but counts by step instead of 1

range(0, 10, 2) → 0, 2, 4, 6, 8
🐍 range() Examples
# Count 0 to 4
for i in range(5):
    print(i)        # 0 1 2 3 4

# Count 2 to 5
for i in range(2, 6):
    print(i)        # 2 3 4 5

# Count by 2 (even numbers)
for i in range(0, 10, 2):
    print(i)        # 0 2 4 6 8

Lesson 4

break & continue

Sometimes you need to control a loop mid-way. break exits the loop completely. continue skips the current step and jumps to the next one.

🛑
break

Stops the entire loop immediately

Like an emergency exit — you leave the loop right away!
for i in range(6):
    if i == 3:
        break
    print(i)
# Output: 0 1 2
⏭️
continue

Skips the current iteration and moves to the next

Like skipping a question on a test — you keep going!
for i in range(6):
    if i == 3:
        continue
    print(i)
# Output: 0 1 2 4 5

Lesson 5

Nested Loops

A nested loop is a loop placed inside another loop. The inner loop runs completely for every single step of the outer loop.

Think of it like a clock: for every 1 turn of the minute hand (outer), the second hand (inner) goes all the way around 60 times!

🐍 Nested Loop Example
subjects = ["Math", "Science"]
days     = ["Mon", "Tue", "Wed"]

for subject in subjects:
    for day in days:
        print(subject, "-", day)

# Math - Mon
# Math - Tue
# Math - Wed
# Science - Mon
# Science - Tue
# Science - Wed

Lesson 6

          
The while Loop

        

      


      

        
A while loop keeps running as long as a condition is True. When the condition becomes False, the loop stops.

        

          

            while
            condition is True
            :
          

          

            →
            run code (update the condition!)
          

        

      


      

        
⚠️ Warning — Infinite Loops! Always make sure your while loop has a way to become False eventually, or it will run forever and crash your program!

      


      

        
🐍 Basic while loop

        
count = 1

while count <= 5:
    print("Count is:", count)
    count += 1   # IMPORTANT: update the variable!

# Count is: 1
# Count is: 2
# Count is: 3
# Count is: 4
# Count is: 5

        
      


      

        
🐍 while with break

        
i = 1

while True:           # Runs forever...
    print(i)
    if i == 3:
        break         # ...until break exits it!
    i += 1

# Output: 1 2 3

Lesson 7

          
else in Loops

        

      


      

        
In Python, both for and while loops can have an else block. This else runs once when the loop finishes normally — but not if the loop was stopped by break.

      


      

        

          
for … else

          

            
for i in range(3):
    print(i)
else:
    print("Loop done!")
# 0 1 2
# Loop done!

          

        

        

          
while … else

          

            
i = 0
while i < 3:
    print(i)
    i += 1
else:
    print("Done!")
# 0 1 2
# Done!

          

        

      


      

        
🐍 else does NOT run if break is used

        
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("This won't print!")   # ❌ skipped!

# Output: 0 1 2

        
      


      

        💡
        
The else clause in loops is a uniquely Python feature — most other languages don't have it. Use it to run cleanup code after a successful loop!
